# Changelog

## [1.0.0] - 2024-01-04

### Added
- Initial release of the Magnet Search Service.
- Support for multiple adapters: APIBay, Nyaa, Sukebei, and Sample.
- Automatic fallback to backup adapters.
- Magnet link parsing support.
- Embedded frontend in Go binary.
- Search history management.
- Health check API with version information.
- Cross-platform build support (Windows, Linux, macOS).
